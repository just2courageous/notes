# Number row characters
* `!` - Exclamation point
* `@` - At symbol
* `#` - Hash tag
* `$` - Dollar sign
* `%` - Percent sign
* `^` - Caret
* `&` - Ampersand, and sign
* `*` - Asterisk
* `(` - Open parenthesis
* `)` - Close parenthesis
